AI Developer Assistant CLI Tool

This tool is a command-line interface application that provides AI-powered code generation, debugging, review, workflow optimization, and documentation search capabilities.

How to Use

  1. Ensure you have Python 3.10+ installed.
  2. Set your OpenAI API key in the environment variable OPENAI_API_KEY.
  3. Run the tool from the command line: 
    python3 tools/ai-developer-assistant.py
  4. Use the following commands inside the CLI:
    • !gen - Generate code based on prompt
    • !debug - Get debugging help
    • !review - Review code from file
    • !optimize - Optimize your workflow (you will be prompted)
    • !exit - Exit the CLI

This CLI tool leverages OpenAI's GPT-4 model and LangChain for advanced AI assistance in software development.
