Word Manipulation Tool

This desktop Python application provides various word manipulation features including text-to-speech, reversing text, word and character counts, case conversions, acronym generation, password generation, and space removal.

How to Use

The tool is a Python Tkinter GUI application. To use it, you need to have Python installed along with the required packages: 

pip install gtts playsound

Save the following Python script as word_manipulation_tool.py and run it with python word_manipulation_tool.py.

You can also use the new web-based Word Manipulation Tool for instant results directly in your browser.
